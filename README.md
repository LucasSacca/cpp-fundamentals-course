# C++ Fundamentals Course

## 📌 Project Overview

This repository contains materials developed during the **2025.2 semester** as part of my Biomedical Engineering undergraduate program. The project was developed as part of an elective course.

The course focused on building a foundation in C++ programming, progressing from basic concepts to object-oriented programming, with a final mini project applying file handling and data processing.

---

## 🎯 Objective

To develop a solid understanding of:

* Basic programming logic
* Control structures
* Functions and procedures
* Memory manipulation and pointers
* File handling
* Object-oriented programming (OOP)

---

## 📚 Course Structure

The repository is organized into lessons, each covering a specific topic:

### Lesson 01 – Basic I/O and Data Types

Introduction to variables, arithmetic operations, and input/output using `scanf` and `printf`.

### Lesson 02 – Conditionals and Loops

Use of `if`, `else`, `for`, and `while` for decision-making and repetition.

### Lesson 03 – Functions and Procedures

Implementation of functions with return values, procedures (`void`), and use of pointers for parameter passing.

### Lesson 04 – Memory and Data Structures

Introduction to pointers, dynamic memory (`new`/`delete`), and basic data structures such as stack and queue.

### Lesson 05 – Error Handling

Basic exception handling using `try`, `catch`, and `throw`.

### Lesson 06 – File Handling

Reading and writing `.txt` files using file streams.

### Lesson 07 – Object-Oriented Programming

Implementation of classes, encapsulation, and methods (getters and setters).

---

## 🧪 Mini Project

A final mini project was developed to apply the concepts learned throughout the course.

The project performs data analysis on a CSV file, including:

* Counting vaccine occurrences
* Identifying single-dose applications
* Calculating sex distribution percentages
* Counting underage individuals
