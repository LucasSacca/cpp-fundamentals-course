# 📊 Mini Project – CSV Data Analysis

This project was developed as a final assignment for my C++ fundamentals course.

## 📌 Objective

To read and analyze data from a CSV file and extract relevant statistics.

## 🔍 Features

* Reads CSV data using file streams (`ifstream`)
* Parses rows using `stringstream`
* Counts:

  * Number of vaccines from ASTRAZENECA/FIOCRUZ
  * Number of single-dose applications
  * Distribution by sex (M/F)
  * Number of underage patients

## 🧠 Notes

Make sure the file `ExampleCSV.csv` is in the same directory.

This project was developed for educational purposes and focuses on applying file handling and data processing concepts in C++.
